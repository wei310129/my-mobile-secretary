# 高難度多領域情境 701–750

本批聚焦權限、隱私、外部操作、失敗重試、回滾與部分成功。皆為 `未開發`。

701. **輸入：** 明天家族聚餐草稿要分享給老婆和外婆，但外婆只需看時間地點，老婆可協助修改；孩子過敏資訊只讓老婆看。請先列分享權限，不要真的傳送，也不能把敏感備註放在所有人的通知。 **原意：** 同一活动分层共享。 **應做：** 查看／编辑／敏感字段权限；预览；零分享。 **主行為：** 家庭權限規劃。
